## Troubleshooting

**"Location permission denied" message?**
- Check browser settings (see guides above)
- Ensure location services are enabled on your device
- Try using "Find Address" button to manually enter location

**Maps not loading?**
- Check internet connection
- Try refreshing the page
- Clear browser cache
- Ensure JavaScript is enabled

**Can't see my property?**
- Zoom in using + button or scroll wheel
- Switch to Satellite or Hybrid view
- Try entering your full address including ZIP code

## Privacy Notice

Your location is only used to:
- Show your property on the map
- Auto-fill your address
- Calculate yard dimensions

Location data is **never stored** or sent to any server. All calculations happen in your browser.
